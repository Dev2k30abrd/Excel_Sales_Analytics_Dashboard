📊 Excel Sales Analytics Dashboard
🧠 Project Overview

This project is an end-to-end Excel Data Analysis & Dashboard project where raw sales data was cleaned, transformed, analysed and converted into an interactive business dashboard.

The goal of this project is to demonstrate real-world Excel skills required for Data Analyst internships.

🎯 Business Problem

Companies generate huge sales data but struggle to extract insights such as:

Which region generates the most revenue?

Which product category is most profitable?

How revenue changes month-by-month?

What is the average profit margin?

This project solves these problems using Excel analytics & dashboarding.

📂 Dataset

The dataset contains sales transactions including:

Order Date

Region

Product Category

Revenue

Profit

Profit Margin

🧹 Data Cleaning Steps (Excel Skills)

Raw data was messy and required cleaning:

✔ Removed duplicates
✔ Fixed date format issues
✔ Created calculated columns
✔ Converted data into Excel Tables
✔ Handled inconsistent formats
